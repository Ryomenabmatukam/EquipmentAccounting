EquipmentAccounting — готовый учебный проект (WinForms + EF Core + LocalDB)

Как запустить:
1) Открыть EquipmentAccounting.sln в Visual Studio 2022+
2) Запустить проект WinFormsUI
3) При первом запуске БД создастся автоматически (EnsureCreated) и заполнит базовые справочники.

База данных:
- LocalDB: (localdb)\MSSQLLocalDB
- Database: EADB
Строка подключения находится в DAL/DLADbContext.cs

Функции:
- Справочники: Подразделения, Сотрудники, Типы оборудования, Лицензии ПО (CRUD)
- Учет: Оборудование (CRUD + закрепление за сотрудником)
- История: автоматически пишется при изменении ответственного (таблица EquipmentMovements), есть форма просмотра
- ПО: установка/удаление лицензий на оборудование (many-to-many)
- Отчеты: оборудование по подразделениям, ПО на компьютерах сотрудника

Статусы оборудования:
0 - В работе
1 - На списании
2 - В ремонте
